# Content Structure Guidelines

## Article Structure

Articles should follow this structure in WordPress:

1. Title: Clear and descriptive
2. Content: Well-formatted using Gutenberg blocks
3. Custom Fields:
   - Difficulty Level
   - Reading Time
   - AI Model (if applicable)
   - Service Provider (if applicable)

## AI Service Structure

Services should include:

1. Basic Information:
   - Title
   - Provider
   - Description
   - Logo

2. Technical Details:
   - Features
   - API Documentation
   - Integration Guide

3. Pricing Information:
   - Tier Name
   - Price
   - Features per tier

## Glossary Terms

Each term should have:

1. Finnish Title
2. English Translation
3. Short Description
4. Detailed Explanation
5. Usage Examples
6. Related Terms
7. External Resources

## Categories and Tags

Implement a clear taxonomy:

- Main Categories:
  - Guides (Oppaat)
  - News (Uutiset)
  - Blog Posts (Blogit)
  - AI Services (Tekoälypalvelut)
  - Tools (Työkalut)

- Tags for:
  - Technology types
  - Difficulty levels
  - Use cases
  - Industries
